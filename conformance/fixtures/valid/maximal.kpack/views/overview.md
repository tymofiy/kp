# Maximal Fixture — Overview

This is a conformance test pack exercising every optional field in PACK.yaml.
All claims are synthetic fixtures designed to test parser edge cases, not real
knowledge assertions.

The pack includes dense and verbose claim syntax, all eight relation types,
versioned claim IDs, cross-pack references, and both evidence formats.
